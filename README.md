# Face Mask Detection Project

## Overview
The Face Mask Detection Project is an innovative application that utilizes computer vision techniques to identify whether individuals are wearing face masks in real-time. This project aims to enhance safety protocols in public spaces during the ongoing global health crisis.

## Features
- **Real-Time Face Mask Detection**: Uses deep learning algorithms to detect faces and determine if they are wearing masks.
- **High Accuracy**: Built with advanced neural networks ensuring high accuracy in mask detection.
- **User-Friendly Interface**: Simple interface for users to upload images or stream video feeds.
- **Multi-Platform Support**: Can be deployed on various platforms including web and mobile applications.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Vort1xss/Face-Mask.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Face-Mask
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the application with the command:
```bash
python app.py
```
